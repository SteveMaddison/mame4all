MULTIPLE ARCADE MACHINE EMULATOR

This is BETA software.
The following is subject to change.

Setup

  1. Run the program once to create the required directories in appdata
  2. Copy your ROM files (zipped) to /media/YOUR_SD_CARD/pandora/appdata/mame4all/roms/
  3. Optional: copy samples (zipped) to /media/YOUR_SD_CARD/pandora/appdata/mame4all/samples/
  4. Optional: copy artwork to /media/YOUR_SD_CARD/pandora/appdata/mame4all/artwork/

Controls

D-Pad:Movement/Joystick
B,X,A,Y,L,R:Buttons 1 thru 6
Keypad A, ".", Shift, Q:Buttons 1 thru 4 for use in rotated mode
Select:Insert coin (player 1)
Start:Start (player 1)
L+R:Pause (in game) or Exit (in menu)
L+R+Start:Exit to menu
L+R+Select:Show FPS
Start+Select:MAME menu
